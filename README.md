# Nadiem Ahmed Portfolio

A clean personal portfolio for Nadiem Ahmed, adapted from Jaidev Jayakumar's `jdsnotes` source with permission.

## Stack

- Next.js 16 App Router
- React 18
- TypeScript
- Tailwind CSS

## Local Development

```bash
npm install
npm run dev
```

The site is structured around three public routes:

- `/` — positioning, proof points, experience, and selected projects
- `/writing` — essays and external writing
- `/reading` — selected books and a placeholder article shelf

Most editable content lives in `content/site.ts`.
